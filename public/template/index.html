In this demo you can explore Twitch dataset and find out the most important analytics from this network. Lets first get some basic knowledge about Twitch streaming platform.




Each user has a channel and the user can be a streamer. A streamer is a user who broadcasts live video. In that video other users can participate by using stream's chat. Streamer speaks in some language, can be a part of some team and plays some game. Streamers can give roles of Editor, Moderator, or VIP to the most valuable members of their community. A moderator helps streamers in managing chat so that it is safe, welcoming, and fun. VIP users have a special VIP badge that allows them to chat without the effect of slow, subscriber-only, or follower-only chat modes. Other users participating in chat that don't have special badge are considered regular chatters.

Here you can check general statistics where you can find out who are top streamers, games, teams, vips or moderators. Besides general statistics, at graph visualization section you can explore some interesting insights which were made using MAGE (Memgraph Advanced Graph Extensions) query modules. With a help from Memgraph, we can now also develop a streaming app. Check out streaming data section and find out how continuous flow of data can be beautifully visualized.
